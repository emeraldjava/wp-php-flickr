Overview
========

This wp-php-flickr wraps the phpFlickr API but uses the default wordpress DB and HTTP classes to cache and communicate with 
the Flickr cloud.

Shortcodes
==========

[wp_flickr_photosets_getphotos photoset_id=xxx]

Change Log
==========
2014.01.25	Add support to list photos from a specific flickr set using shortcode [wp_flickr_photosets_getphotos photoset_id=xxx].
